PROJECT OVERVIEW
================================================================================
AnnotateAI is a powerful, modern web-based image annotation tool built with 
React and Vite. It allows users to annotate images with bounding boxes, add 
custom labels, and export annotations as JSON data.

Version: 1.0.0
Built with: React + Vite
Created: April 1, 2026

================================================================================

2. Install: cd Image-Annotator-01-04-2026- && npm install

KEY FEATURES
================================================================================

IMAGE MANAGEMENT:
  - Upload single or multiple images
  - Supports: PNG, JPG, WEBP, GIF
  - Auto-displays file size and annotation count
  - One-click image removal

ANNOTATION TOOLS:
  - Click and drag to draw boxes
  - Real-time preview while drawing
  - Minimum box size: 6x6 pixels
  - Crosshair cursor for precision

LABEL MANAGEMENT:
  - Auto-generated names ("Object 1", "Object 2", etc.)
  - Click to rename any label
  - Display box coordinates (X, Y, Width, Height)
  - Delete individual or all annotations

DATA EXPORT:
  - Export as JSON format
  - Includes coordinates rounded to integers
  - File format: annotations.json
  - Contains image name, label, and box dimensions

INTERFACE:
  - Three-column responsive layout
  - Dark theme with amber accents
  - Real-time update on all changes
  - Status bar with helpful information


EXPORTED JSON FORMAT
================================================================================

File name: annotations.json

Structure:
[
  {
    "name": "image_filename.jpg",
    "annotations": [
      {
        "id": "unique_identifier",
        "label": "Object Name",
        "x": 150,
        "y": 200,
        "width": 120,
        "height": 80
      },
      {
        "id": "another_id",
        "label": "Another Object",
        "x": 300,
        "y": 100,
        "width": 200,
        "height": 350
      }
    ]
  }
]

Note: All coordinates are rounded to integers for consistency

TROUBLESHOOTING
================================================================================

PROBLEM 1: Port Already in Use
SOLUTION: Use different port with:
  npm run dev -- --port 3000

PROBLEM 2: Blank Page After Start
SOLUTIONS:
  - Clear browser cache (Ctrl+Shift+Delete)
  - Check browser console for errors (F12)
  - Verify internet connection for font loading
  - Reload page (Ctrl+R or Cmd+R)

PROBLEM 3: Module Not Found Errors
SOLUTION: Reinstall dependencies:
  rm -rf node_modules package-lock.json
  npm install

PROBLEM 4: Slow Performance
SOLUTIONS:
  - Use images smaller than 5MB
  - Close other applications
  - Clear browser cache
  - Check available RAM

PROBLEM 5: npm install Takes Too Long
SOLUTION: Use npm cache clean:
  npm cache clean --force
  npm install

PROBLEM 6: Cannot Find Node or npm Commands
SOLUTION: 
  - Reinstall Node.js from nodejs.org
  - Restart terminal/command prompt
  - Add Node to system PATH (Windows): 
    Control Panel > System > Environment Variables

TIPS & BEST PRACTICES
================================================================================

ANNOTATION BEST PRACTICES:
- Use descriptive, consistent label names
- Draw boxes tightly around objects
- Keep box sizes proportional for ML training
- Export frequently to avoid losing work

KEYBOARD SHORTCUTS:
- Escape: Cancel label edit
- Enter: Confirm label changes
- Reload page: Ctrl+R (Chrome/Firefox)

PERFORMANCE TIPS:
- Limit images to 10-20 at a time for smooth performance
- Use compressed images when possible
- Close unused browser tabs

FILE MANAGEMENT:
- Keep backup copies of exported JSON files
- Use version control for collaborative work
- Name exported files with dates: annotations_04-01-2026.json
